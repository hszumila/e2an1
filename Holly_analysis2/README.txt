Codes for the neutron study:
1. calcNCorr.cpp : finds the path length correction, calculates correction functions and resolution
2. plotNeff.cpp : plots the neutron efficiency
3. findSRCcuts.cpp : finds the optimal SRC cuts for the neutron resolution
4. findMFcuts.cpp: finds the optimal MF cuts for the neutron resolution
5. ratioNP_MF.cpp : calculates the n/p ratio for low momentum
6. ratioNP_SRC.cpp : calculates the n/p ratio for high momentum
